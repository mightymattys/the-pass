# The Pass v0.7.1

`v0.7.1` is a release-process fix-forward for `v0.7.0`.

- SHA-256 entries use downloaded asset basenames and verify with standard tooling.
- Release audit and release-note assets are resolved dynamically from the tag version.
- Framework code, candidate gate semantics, and the locked live boundary are unchanged.

The complete implementation and safety changes remain documented in the `v0.7.0` notes and
the `v0.7.1` release audit.
